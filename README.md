# bluetooth-tui2

Bluetooth control TUI in Go using Bubble Tea.

## Run

```bash
go run ./cmd/bluetooth-tui
```

Controls:

- `j` / `k` or arrows: move selection
- `enter`: pair/connect selected device
- `r`: re-scan
- `q`: quit

## Test

```bash
go test ./...
```

Box at top with
j/k or arrows Nav, enter symbol connect, q quit, r rescan,
highlight keys
